    / ABOUT THE LAB

    Learning by building—and recording the whole process.

    Maker-Lab is a personal workspace for experimenting with hardware and software, learning embedded systems, and turning rough prototypes into presentable projects.

    Each build is a chance to record the wiring, code, problems, and solutions that make the next experiment better. The repository will grow over time alongside the skills behind it.
